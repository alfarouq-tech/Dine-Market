## Getting Started
To get a local copy up and running follow these simple example steps.
  - Click on `code` button
  - Copy HTTPS link
  - Open command
  - Type `git clone \`HTTPS link`\`

### Run the app
  * npm run start

### Deployment
  * npm run build
